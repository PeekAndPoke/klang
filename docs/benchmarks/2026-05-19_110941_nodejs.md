# Effect Benchmark Results

- **Platform:** JS / Node.js/24
- **Sample rate:** 44100 Hz
- **Block size:** 128 frames
- **Warmup:** 10000 blocks

RTF = render time / audio time. Lower is better (RTF < 1.0 = faster than real-time).
