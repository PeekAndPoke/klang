#### Pitch Envelope

- [x] **`pattack()`** ‚Äî *Attack time of the pitch envelope* (synonym: `patt`). Controls how quickly pitch ramps from its
  start point toward the envelope peak when a note begins; shorter times create a snappier ‚Äúthwip,‚Äù longer times make a
  slow bend‚Äëup. [[1]](https://strudel.cc/learn/effects/)
- [x] **`pdecay()`** ‚Äî *Decay time of the pitch envelope* (synonym: `pdec`). After the peak, this sets how fast pitch

  glide. [[1]](https://strudel.cc/learn/effects/)
- [x] **`prelease()`** ‚Äî *Release time of the pitch envelope* (synonym: `prel`). Controls how pitch returns when the
  note ends; longer release makes the tail drift back, shorter release snaps it
  off. [[1]](https://strudel.cc/learn/effects/)
- [x] **`penv()`** ‚Äî *Pitch‚Äëenvelope amount* (synonym: `pamt`). Sets the total pitch modulation depth (in semitones);
  higher values give dramatic swoops, lower values add subtle pitch movement. [[1]](https://strudel.cc/learn/effects/)
- [x] **`pcurve()`** ‚Äî *Pitch‚Äëenvelope curve shape* (synonym: `pcrv`). Controls how the pitch change is distributed over
  time (e.g., more linear vs. more exponential), shaping the feel of the bend. [[1]](https://strudel.cc/learn/effects/)
- [x] **`panchor()`** ‚Äî *Pitch‚Äëenvelope range anchor* (synonym: `panc`). Sets where the envelope is centered within the
