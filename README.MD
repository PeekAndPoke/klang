## Modules

| Module            | Platform  | Description                                                                    |
|-------------------|-----------|--------------------------------------------------------------------------------|
| `klang`           | JS/JVM    | Main application вҖ” web UI, pages, routing, audio player integration            |
| `klangscript`     | Common    | Scripting language вҖ” hand-rolled parser, tree-walking interpreter, type system |
| `klangscript-ui`  | JS        | CodeMirror editor with code completion, hover docs, tool badges                |
| `klangjs`         | JS        | External JS declarations (CodeMirror bindings) вҖ” no project dependencies       |
| `klangui`         | JS        | Shared UI components вҖ” theme, hover popups, tool registry, SVG helpers         |
| `klangblocks`     | Common/JS | Visual block editor вҖ” AST-to-blocks, code generation, round-trip editing       |
| `strudel`         | Common/JS | Pattern language вҖ” Kotlin port of strudel.cc for live coding music             |
| `strudel-ksp`     | JVM       | KSP annotation processor вҖ” extracts DSL documentation from KDoc                |
| `audio_bridge`    | Common    | Shared audio data types вҖ” voice data, playback signals, audio config           |
| `audio_be`        | Common    | Audio backend вҖ” oscillators, filters, effects, voice rendering                 |
| `audio_fe`        | Common/JS | Audio frontend вҖ” playback scheduling, sample loading, engine facade            |
| `audio_jsworklet` | JS        | WebAudio worklet вҖ” runs audio rendering in a dedicated thread                  |
| `common`          | Common    | Shared utilities вҖ” caching, math helpers, platform abstractions                |
| `tones`           | Common    | Music theory вҖ” note names, scales, frequencies, MIDI conversion                |
| `klang-notebook`  | Common/JS | Interactive notebook support вҖ” cell execution, reactive evaluation             |
