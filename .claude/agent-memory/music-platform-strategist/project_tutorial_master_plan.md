Genre-integrated tutorial master plan completed 2026-04-07. Builds on gap analysis.

Key decisions:

- Every new tutorial gets a unique genre assignment (36 new genres, zero repeats)
- Genre assignments are NATURAL fits (e.g., Euclidean = Afrobeat, Filter Envelopes = Acid House, Swing = Boom-Bap)
- Critical user requirement: tutorials must teach AND demonstrate Klang's versatility
- Full library spans 55+ genres across 6 continents of music

Phase 1 genre assignments (beginner, make first):

1. H1 Signals → Synthpop
2. H2 Randomness → Minimal Techno
3. H7 Probability → Jazz
4. H4 Swing → Boom-Bap Hip-Hop
5. H5 Euclidean → Afrobeat
6. H3 Samples → Trip-Hop
7. H10 Mini-notation → UK Garage / 2-Step
8. H6 struct/mask → Reggaeton
9. H8 Orbits → Dub Reggae
10. H9 Pattern Math → Neo-Soul

**Why:** User explicitly stated tutorials must spread across maximum genres to demonstrate versatility. Existing 37
tutorials cluster heavily in Ambient/Electronica/IDM. New tutorials deliberately cover Hip-Hop, Latin, African,
Caribbean, Rock, Classical, Folk, and more.

**How to apply:** When creating any new tutorial, check this plan for its assigned genre. The genre is not wallpaper —
it's the reason the concept is being taught through that musical context. The jingle/example code must authentically
represent the genre.
